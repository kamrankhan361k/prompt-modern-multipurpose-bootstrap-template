Introduction

Prompt is a fully featured moderen multi purpose template built on top of awesome Bootstrap 5.1.3. It's for building responsive, mobile-first landing pages, websites with Bootstrap.

This documentation is guidelines for all users including beginner, experienced web developer, etc

We really care for our buyers and so in case if you are having any question or feedback, please feel free to get back to us via email support@coderthemes.com or by filling out the contact form on our website.

Folder Structure

Extract the zip file you received after purchase and you would find the exact below files and folders: 

├── prompt
│   ├── dist - auto generated production ready files, including HTML, css, js, images, etc
│   ├── src/
│   │   └── assets - the reusable assets used in different demos
│   │   │   └── fonts
│   │   │   └── images
│   │   │   └── js
│   │   │   └── scss
│   │   └── docs - source code for documentation
│   │   └── partials - partial templates - included in all the pages. e.g. top navbar, heros, etc
│   ├── gulpfile.js - The build commands for Gulp
│   ├── package.json
└── README.md
